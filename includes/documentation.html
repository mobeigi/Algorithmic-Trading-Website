Documentation

Using our software

  1. Download the program here (or refer to the downloads tab above). Make sure you choose the link corresponding to your operating system.
  2. Unzip the downloaded file by right-clicking and selecting 'Extract here'.

  3. Open the 'params file' text file and configure the following fields:

    • input_csvFile
    • output_logFile
    • output_csvFile

    Refer to the 'Configuring params file' section below for more details.

  4. *Some steps about the make file

  5. Open command line (or command prompt), and run our program by entering the following command:

    %./trockAT paramaters.param

Integration of our software in your application

  1. Make sure you have downloaded the program, and configured the 'params file' correctly (ensuring the links for the input csv file, and directories for output csv files are correct with your program).
  2. Make your program run our software using the command stated above through command line.
  3. Read the output files (located in the path(s) you configured for the output files) for the trading data, and let your program process the information as necessary.

Configuring the 'Params file'

There are a variety of fields that you are able to customise.

Field Description
input_csvFile Link to the input csv file. The file is respective to the directory of where the trockAT application is located
output_logFile Directory for the output log file
output_csvFile Directory for the file containing orders
threshold A higher threshold will result in the strategy to generate less buy/sell signals
returnsInCalculation Number of days that a trade has occurred used to calculate the last moving average
