Using our trading module

  1. Download the standalone module here (or refer to the downloads tab above). Make sure you choose the link corresponding to your operating system.
  2. Unzip the downloaded file by right-clicking and selecting 'Extract here'.

  3. Open the 'params file' text file and configure the following fields:

    • input_csvFile
    • threshold
    • returnsInCalculation
    • startDate
    • endDate

    Refer to the 'Configuring params file' section below for more details.

  4. Open command line (or command prompt), and run our module by entering the following command:

    %trockAT paramaters.param

    For users running the Linux version, the command will be:

    %./trockAT paramaters.param

Integration of our trading module in your application

  1. Make sure you have downloaded the module, and configured the 'params file' correctly (ensuring the links for the input csv file, and directories for output csv files are correct with your program).
  2. Make your program run our module using the command stated above through command line.
  3. Read the output files (located in the path(s) you configured for the output files) for the trading data, and let your program process the information as necessary.

Using our Trading Platform

  1. Download the Testing Platform from the downloads tab.
  2. Unzip the downloaded file by right-clicking and selecting 'Extract here'.

  3. Double click the executable file.

Configuring the 'Params file'

There are a variety of fields that you are able to customise.

Field Description
input_csvFile* Direct link to the input csv file. (Default value is linked to 'sample_csv_file' as the input)
output_logFile Direct link for the output log file.
output_csvFile Direct link for the file containing orders
threshold* A higher threshold will result in the strategy to generate less buy/sell signals. (Default value is 0.05)
returnsInCalculation* Number of days that a trade has occurred used to calculate the last moving average. (Default value is 5)
startDate, endDate* The period when trading can occur. Each date must be expressed in the format 'DD-MMM-YYYY'.

* this field cannot be empty

Common Issues

Q: I'm getting a "bash: ./[executable name]: Permission denied" error message on Linux.

A: The program is currently not allowed to be executed. To fix this, simply run the following command:

%chmod 755 [executable name]
